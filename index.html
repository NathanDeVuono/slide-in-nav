Slide-in Navigation

I saw a little demo online that had an effect like this using jQuery to render menus that slide or fold in from off the rendered area, and I thought to myself, "Why not make it work in CSS". Turns out it was really simple!

All it really takes is:

nav { height:0; overflow:hidden; transition:.7s ease height; }
.hoverme { background:orange; &:hover nav { height:20px; } }

Neat, huh?

The same effect could be achieved using transition: to actually move the nav element onto the screen, but for the sake of lazy, I just adjusted the height.
